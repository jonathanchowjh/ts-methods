# ts-utils

### PreInstallation

- Configure your ssh permissions if you are downloading a private repo

### Usage (command line)

- update `shell/scripts.command` line 2 with `cd ${pwd_PATH}`
- run `chmod +x shell/scripts.command`
- double click `shell/scripts.command`

### Usage (command line)

```
npm run scripts
```
